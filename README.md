NextLayer - https://alexkrussell.github.io/NextLayer/
Overview 
NextLayer is a 3D Printing Digital Marketplace company website designed to showcase professional web design and product licensing. The site highlights licensing services, project for purchase, and contact information through a responsive and user-friendly interface. 
Project Purpose 
The goal of this project was to: 
•	Create a professional business website
•	Demonstrate front-end development skills
•	Practice responsive web design 

Features 
•	Responsive design for desktop, tablet, and mobile devices
•	Modern navigation menu
•	Hero section with company branding
•	Services section
•	About Us section
•	Portfolio/Projects showcase
•	Contact form
•	Professional layout and styling
•	Interactive user experience
